# inventory-Management-AI-Agent
The Inventory Management AI Agent is a conversational assistant designed to simplify and automate inventory tracking. Activated directly via chat, the agent allows users to effortlessly manage stock updates and queries in real time.


🔹 Key Features
Conversational Interface: Interact with the agent via natural language in a chat interface — no need to learn commands or open spreadsheets.

Google Sheets Integration: Seamlessly syncs with a Google Sheet to reflect inventory changes based on your inputs.

Dynamic Updates: Record purchases and sales by simply telling the agent — it will update the inventory for you.

Real-Time Status Checks: Get live information on current stock levels for any item.


You can interact with agent just like whatsapp messages and get you inventory managed by it.
